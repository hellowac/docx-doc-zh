WD_BUILTIN_STYLE

别名：WD_STYLE

指定内置的 Microsoft Word 样式。

例子：

from docx import Document
from docx.enum.style import WD_STYLE

document = Document()
styles = document.styles
style = styles[WD_STYLE.BODY_TEXT]
